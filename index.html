Featured Research Project

Neural Traffic Routing and Congestion Optimisation System

This research investigates whether adaptive congestion-aware routing can improve urban traffic efficiency compared to traditional shortest-path algorithms.

Methods

  • OpenStreetMap Road Networks
  • OSMnx
  • NetworkX
  • Dijkstra Search
  • A* Search
  • Adaptive Congestion Weighting
  • Custom Neural Network Built from Scratch
  • Large-Scale Traffic Simulation

Technical Contributions

  • Custom feed-forward neural network implementation using NumPy
  • Dynamic congestion modelling
  • Adaptive edge-weight updates
  • Vehicle rerouting simulation
  • Performance evaluation framework
  • Comparative algorithm analysis
